# DNA Sequence Analyzer 

A Python bioinformatics tool that analyzes DNA sequences.

## Features
- Nucleotide frequency analysis
- GC and AT content calculation
- DNA → RNA transcription
- RNA → Protein translation
- Reverse complement generation
- ORF detection
- Motif searching
- Melting temperature estimation
- FASTA file support

## How to Run

1. Download `dna_analyzer.py` and your `.fasta` file into the same folder
2. Run:
```
python3 dna_analyzer.py
```

## Example
This project uses the human **FSHB** (Follicle Stimulating Hormone Beta Subunit) mRNA sequence from NCBI.
